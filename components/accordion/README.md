# Accordion v1.0

**Status:** Production Stable

## Purpose

Reveal supporting information while maintaining page context.

## Motion

- One panel open at a time
- Editorial reveal rather than content dump
- Upward motion for interactive elements
- Opacity-based reveal for static content
- Left-to-right reading rhythm

## Dependencies

- GSAP
- js-accordion
- js-accordion-item
- js-accordion-trigger
- js-accordion-panel
- js-accordion-status
