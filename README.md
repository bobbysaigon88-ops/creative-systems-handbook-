# Creative Systems Handbook

The canonical source of truth for the SonTran.Design Creative Systems Handbook (CSH) — a living design system documenting principles, components, motion language, utilities, and implementation patterns.

## Structure

- `philosophy/` — design principles and system thinking
- `components/` — reusable UI components
- `utilities/` — utility classes and usage rules
- `tokens/` — design tokens and variables
- `examples/` — implementation examples and prototypes
